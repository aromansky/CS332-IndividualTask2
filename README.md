# Ray Tracing Engine — Cornell Box

Простой рейтрейсинг-движок, реализованный на **C#**, предназначенный для визуализации классической сцены **Cornell Box**.  
Проект демонстрирует основные принципы трассировки лучей, работу с освещением, тенями и материалами.

---

## Возможности

### Сцена
- Полностью замкнутая **Cornell Box**
- Все стены участвуют в расчёте освещения
- Поддержка объектов:
  - Кубы
  - Сферы

---

### Освещение
- Точечные источники света
- Теневые лучи для построения реалистичных теней
- Диффузное освещение по модели Ламберта

---

### Материалы
- Матовые поверхности
- Зеркальные отражения
- Прозрачные объекты с преломлением света
- Настраиваемый индекс преломления

---

### Алгоритм
- Итеративная трассировка лучей
- Контроль глубины отражений и преломлений
- Многопоточный рендеринг для ускорения вычислений

---

### Управление
- Интерактивное изменение свойств материалов
- Возможность сделать стены зеркальными
- Управление объектами в сцене
- Отображение прогресса рендеринга в реальном времени
